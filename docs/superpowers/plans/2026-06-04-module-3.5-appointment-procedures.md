# Module 3.5 — Per-Appointment Procedures Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Let staff record the procedures performed at an appointment (CDT code, tooth/surface, fee, optional manual insurance/patient estimate) and view a per-patient procedure history.

**Architecture:** Two new tables — `cdt_codes` (seeded reference data) and `appointment_procedures` (PHI, mirrors `treatment_plan_items`). A patient/appointment-scoped FastAPI router follows the existing `treatment_plans.py` pattern (practice-scope + write-role guards, PHI audit touch, idempotency middleware). Zod schemas in `packages/shared-types` generate the Pydantic models. Two React surfaces: a Procedures section in `AppointmentModal` and a read-only Procedure History tab on the patient chart.

**Tech Stack:** FastAPI + SQLAlchemy 2.0 (async) + Alembic, Pydantic v2 (generated from Zod), Postgres, Next.js (App Router) + TypeScript, pytest (`-m integration`), Vitest.

**Spec:** `docs/superpowers/specs/2026-06-04-module-3.5-appointment-procedures-design.md`
**Build order context:** `docs/superpowers/specs/phase3-build-order.md`

## Key codebase facts (verified)

- Mixins in `apps/api/app/models/base.py`: `PHIMixin` (adds `last_accessed_by`, `last_accessed_at` + timestamps + UUID PK) for PHI tables; `TimestampMixin` for non-PHI. `cdt_codes` → `TimestampMixin`; `appointment_procedures` → `PHIMixin`.
- `Appointment.patient_id` is **nullable** (`apps/api/app/models/appointment.py:29`). A procedure requires a patient — POST must 422 if the appointment has no `patient_id`.
- Guards live in `patients.py`: `_require_practice_scope(request) -> uuid.UUID` and `_require_write_role(request)`. Import them like `treatment_plans.py` does: `from app.routers.patients import _require_practice_scope, _require_write_role`. Write roles: `admin`, `provider`, `front_desk`.
- PHI audit pattern: a `_phi_audit_*` helper issues an `UPDATE ... SET last_accessed_by=..., last_accessed_at=now()` then `session.commit()`. Copy `_phi_audit_plan` from `treatment_plans.py:109`.
- Schemas are **generated**: author Zod in `packages/shared-types/src/schemas/`, then run `pnpm generate` (turbo) from repo root → writes `apps/api/app/schemas/generated.py`. Never hand-edit `generated.py`. Field aliases are camelCase (e.g. `fee_cents` ↔ `feeCents`).
- Error shape: `HTTPException(status_code=..., detail=ApiError(error=Error(code=..., message=...)).model_dump(by_alias=True))`.
- Migrations: `apps/api/alembic/versions/`, latest is `0024`. New revision `0025`, `down_revision = "0024"`. Models must be imported in `apps/api/alembic/env.py` for autogenerate (we hand-write DDL here, but still register the models).
- Tests: integration in `apps/api/tests/integration/` (`-m integration`, needs Postgres+Redis); fixtures `client`, `auth_headers`, `practice`, `db_session`, and `mut(headers)` (adds a fresh `Idempotency-Key`) come from `tests/integration/conftest.py`. Router/unit tests in `apps/api/tests/routers/`.
- Frontend: API client modules in `apps/web/lib/api/` (e.g. `scheduling.ts`); appointment modal `apps/web/components/scheduling/AppointmentModal.tsx`; patient chart tabs mounted in `apps/web/app/(app)/patients/[patientId]/page.tsx`, tab components like `apps/web/components/patients/TreatmentPlanTab.tsx`.

from __future__ import annotations

import logging
import uuid
from datetime import UTC, datetime
from typing import Any

from fastapi import APIRouter, HTTPException, Request
from sqlalchemy import func, or_, select, update
from sqlalchemy.ext.asyncio import AsyncSession

from app.core.db import get_session_factory
from app.models.appointment import Appointment as AppointmentModel
from app.models.appointment_procedure import AppointmentProcedure as ProcedureModel
from app.models.appointment_procedure import CdtCode as CdtCodeModel
from app.routers.patients import _require_practice_scope, _require_write_role
from app.schemas.generated import (
    ApiError,
    AppointmentProcedure,
    AppointmentProcedureListResponse,
    CdtCode,
    CreateAppointmentProcedure,
    Error,
    ProcedureTotals,
    UpdateAppointmentProcedure,
)

logger = logging.getLogger(__name__)

# Appointment-scoped router for procedures.
router = APIRouter(
    prefix="/api/v1/appointments/{appointment_id}/procedures",
    tags=["appointment-procedures"],
)

# Practice-wide CDT typeahead router.
cdt_router = APIRouter(prefix="/api/v1/cdt-codes", tags=["appointment-procedures"])

_CDT_SEARCH_LIMIT = 25


def _err(code: str, message: str) -> dict[str, Any]:
    return ApiError(error=Error(code=code, message=message)).model_dump(by_alias=True)


def _cdt_to_schema(row: CdtCodeModel) -> CdtCode:
    return CdtCode(
        id=row.id,
        code=row.code,
        description=row.description,
        category=row.category,
        defaultFeeCents=row.default_fee_cents,
        isActive=row.is_active,
    )


@cdt_router.get("", response_model=list[CdtCode])
async def search_cdt_codes(request: Request, q: str | None = None) -> list[CdtCode]:
    _require_practice_scope(request)
    async with get_session_factory()() as session:
        query = select(CdtCodeModel).where(
            CdtCodeModel.is_active.is_(True),
            CdtCodeModel.deleted_at.is_(None),
        )
        if q:
            term = q.strip()
            query = query.where(
                or_(
                    CdtCodeModel.code.ilike(f"{term}%"),
                    CdtCodeModel.description.ilike(f"%{term}%"),
                )
            )
        query = query.order_by(CdtCodeModel.code.asc()).limit(_CDT_SEARCH_LIMIT)
        rows = (await session.scalars(query)).all()
    return [_cdt_to_schema(r) for r in rows]
```

async def _load_scoped_appointment(
    session: AsyncSession, appointment_id: uuid.UUID, practice_id: uuid.UUID

    appt = await session.scalar(
        select(AppointmentModel).where(
            AppointmentModel.id == appointment_id,
            AppointmentModel.practice_id == practice_id,
            AppointmentModel.deleted_at.is_(None),
        )

        raise HTTPException(
            status_code=404, detail=_err("APPOINTMENT_NOT_FOUND", "Appointment not found")

@router.get("", response_model=AppointmentProcedureListResponse)
async def list_procedures(
    appointment_id: uuid.UUID, request: Request
) -> AppointmentProcedureListResponse:
    practice_id = _require_practice_scope(request)
    user_sub = getattr(request.state.user, "sub", None)

    async with get_session_factory()() as session:
        await _load_scoped_appointment(session, appointment_id, practice_id)

        rows = (
            await session.scalars(
                select(ProcedureModel)
                .where(
                    ProcedureModel.appointment_id == appointment_id,
                    ProcedureModel.deleted_at.is_(None),
                )
                .order_by(ProcedureModel.created_at.asc())
            )
        ).all()

        for r in rows:
            await _phi_audit_procedure(session, r.id, user_sub)
        if rows:
            await session.commit()

    fee_total = sum(r.fee_cents for r in rows)
    ins_total = sum(r.insurance_est_cents or 0 for r in rows)
    pat_total = sum(r.patient_est_cents or 0 for r in rows)

    return AppointmentProcedureListResponse(
        items=[_proc_to_schema(r) for r in rows],
        totals=ProcedureTotals(
            feeCentsTotal=fee_total,
            insuranceEstCentsTotal=ins_total,
            patientEstCentsTotal=pat_total,
        ),
    )

def _resolve_code_fields(body: CreateAppointmentProcedure) -> str | None:
    """Return procedure_code or raise 422 when neither code source is present."""
    if body.cdt_code_id is None and not body.procedure_code:
        raise HTTPException(
            status_code=422,
            detail=_err(
                "PROCEDURE_CODE_REQUIRED",
                "Either cdtCodeId or procedureCode must be provided",
            ),
        )
    return body.procedure_code


@router.post("", status_code=201, response_model=AppointmentProcedure)
async def create_procedure(
    appointment_id: uuid.UUID, body: CreateAppointmentProcedure, request: Request
) -> AppointmentProcedure:
    practice_id = _require_practice_scope(request)
    _require_write_role(request)
    user_sub = getattr(request.state.user, "sub", None)

    procedure_code = _resolve_code_fields(body)

    async with get_session_factory()() as session:
        appt = await _load_scoped_appointment(session, appointment_id, practice_id)
        if appt.patient_id is None:
            raise HTTPException(
                status_code=422,
                detail=_err(
                    "APPOINTMENT_HAS_NO_PATIENT",
                    "Cannot add procedures to an appointment without a patient",
                ),
            )

        # If cdt_code_id given, validate it exists and is active.
        if body.cdt_code_id is not None:
            cdt = await session.scalar(
                select(CdtCodeModel).where(
                    CdtCodeModel.id == body.cdt_code_id,
                    CdtCodeModel.deleted_at.is_(None),
                )
            )
            if cdt is None:
                raise HTTPException(
                    status_code=422, detail=_err("CDT_CODE_NOT_FOUND", "Unknown CDT code")
                )
            if not procedure_code:
                procedure_code = cdt.code

        row = ProcedureModel(
            id=uuid.uuid4(),
            practice_id=practice_id,
            appointment_id=appointment_id,
            patient_id=appt.patient_id,
            cdt_code_id=body.cdt_code_id,
            procedure_code=procedure_code,
            procedure_name=body.procedure_name,
            tooth_number=body.tooth_number,
            surface=body.surface,
            fee_cents=body.fee_cents,
            insurance_est_cents=body.insurance_est_cents,
            patient_est_cents=body.patient_est_cents,
            estimate_source=body.estimate_source,
            notes=body.notes,
            last_accessed_by=user_sub,
            last_accessed_at=datetime.now(UTC),
        )
        session.add(row)
        await session.commit()
        await session.refresh(row)

    logger.info("Procedure added: appointment_id=%s procedure_id=%s", appointment_id, row.id)
    return _proc_to_schema(row)
```

> Note on the 422 "code or name" test: the Pydantic model requires `procedureName` (min length 1) but allows missing `procedureCode`/`cdtCodeId`. `_resolve_code_fields` enforces the at-least-one-code rule and returns 422. The test payload omits both code fields → 422. Good.

```python
patient_router = APIRouter(
    prefix="/api/v1/patients/{patient_id}/procedures",
    tags=["appointment-procedures"],
)


@patient_router.get("", response_model=AppointmentProcedureListResponse)
async def list_patient_procedures(
    patient_id: uuid.UUID, request: Request
) -> AppointmentProcedureListResponse:
    practice_id = _require_practice_scope(request)
    user_sub = getattr(request.state.user, "sub", None)

    async with get_session_factory()() as session:
        rows = (
            await session.scalars(
                select(ProcedureModel)
                .where(
                    ProcedureModel.patient_id == patient_id,
                    ProcedureModel.practice_id == practice_id,
                    ProcedureModel.deleted_at.is_(None),
                )
                .order_by(ProcedureModel.created_at.desc())
            )
        ).all()
        for r in rows:
            await _phi_audit_procedure(session, r.id, user_sub)
        if rows:
            await session.commit()

    fee_total = sum(r.fee_cents for r in rows)
    ins_total = sum(r.insurance_est_cents or 0 for r in rows)
    pat_total = sum(r.patient_est_cents or 0 for r in rows)
    return AppointmentProcedureListResponse(
        items=[_proc_to_schema(r) for r in rows],
        totals=ProcedureTotals(
            feeCentsTotal=fee_total,
            insuranceEstCentsTotal=ins_total,
            patientEstCentsTotal=pat_total,
        ),
    )
```

Register in `main.py`: `app.include_router(appointment_procedures.patient_router)`.

> Spec says sort by appointment `start_time` desc. Procedure `created_at` is a faithful proxy for visit order in 3.5 and avoids a join; if true `start_time` ordering is needed later, add a join to `appointments`. Documented intentional simplification.
