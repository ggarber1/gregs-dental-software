## Features

### Scheduling
- Drag-and-drop scheduling
- Appointment mirroring (single appointment across multiple operatories, no duplication)
- Web Scheduling: new patient, existing patient, recall, ASAP list
- ODTouch tablet app for chairside scheduling and charting
- Automated appointment confirmations and reminders
- Mobile schedule access via ODMobile

### Clinical
- Graphical tooth chart with dynamic visualization
- Perio charting with Bola AI voice perio integration (hands-free dictation)
- Treatment planning
- Electronic prescriptions (eRx)
- Medical/health histories
- eForms (electronic patient intake forms)
- eClipboard: digital check-in (v25 has dark mode, accessibility options)
- eClipboard Web: patients fill forms on their own device before arrival

### Billing
- Accounts receivable management
- Electronic claims submission
- Claim attachments via EDS/DentalXChange
- Recurring charges / membership plan billing with automated CC processing
- Message-to-Pay: text/email patient a payment link to the Patient Portal (v24.1, 2024)
- Insurance eligibility verification
- Frequency limitation checks with customizable Code Groups, Treatment Areas, Age Limits

### Imaging
- Imaging Module with BetterDiagnostics integration — AI-annotated radiographs
- Pearl AI integration for radiograph analysis (v24.3+, Dec 2024)
- Imaging bridges for major hardware: Acteon, ADSTRA, Apteryx, BioPAK, Carestream, Dexis, and many others

### Reporting
- 50+ built-in standard reports (production, income, A/R, broken appointments, new patients)
- Graphical/visual trend reports
- **User Query tool: write custom SQL directly against the database** — a unique differentiator
- Custom .rdl report files
- Customizable report favorites and security permissions per user group

### Patient Engagement
- Patient Portal (appointments, statements, treatment plans, secure messaging, payments)
- Automated reminders, recall messages, thank-you messages
- Integrated two-way texting
- Secure email
- Web forms

### Mobile Access
- ODMobile: view schedule, patient info, pharmacy info from home
- Mobile Web: browser-based schedule and patient record access
- ODTouch: tablet app for clinical/chairside use
- eClipboard Web: patient-facing web form completion on personal devices

## Target Market
- **Primary:** Solo and small group general dental practices (1–10 employees; $1M–$10M revenue modal)
- **Secondary:** Multi-location groups and DSOs wanting affordability + customizability
- **Tech-savvy practices** wanting SQL access, custom integrations, and maximum data control
- **Geographic:** Primarily US; Canadian market is underserved
- **Specialty:** Supported but not purpose-built; some specialty workflows unsupported

| Aspect | Details |

## User Reviews

### Praise
- **Lowest cost** among full-featured competitors
- Highly customizable — workflows built around the practice
- **Custom SQL reports are powerful** for data-savvy offices
- Customer support (phone/chat) consistently rated excellent and responsive
- **Transparent, published pricing** — a genuine differentiator
- Large third-party ecosystem — plug in best-in-class tools
- Long-term users call it "the most thorough" option

### Complaints
- **Not cloud-native by default** — significant IT overhead required
- **Steep learning curve** for front office and billing staff
- **Open Dental Cloud glitches** — some users report reopening app 20+ times/day
- Laggy performance in remote/VPN setups at large organizations
- **Patient tracking is weak** — tracking lost/lapsed patients requires manual SQL queries
- **Canadian support minimal** — many features US-only
- UI is dated and not consumer-friendly
- After-hours support costs extra

## Key Weaknesses Summary

1. **Not cloud-native** — IT overhead; server required or third-party hosting cost
2. **Dated UI/UX** — looks and feels like 2010s desktop software
3. **Patient tracking is poor** — no proactive lost-patient identification; needs SQL
4. **Canadian/international neglect** — many features US-only
5. **Fragmented experience** — core software + eServices + third-party integrations = multiple vendors and bills
6. **Specialty depth is limited** — not purpose-built for ortho, oral surgery, advanced perio
7. **After-hours support costs extra**
8. **Open Dental Cloud reliability issues** — freezing reported in early cloud version
9. **Steep learning curve** — especially for front office and billing staff
